Advanages of JavaScript

  • Less server interaction – we can valiadate user input before senting the page of to the server. this saves server traffic, which means less load on our server.
  • Immediate feedback to the visitors – they do not have to wait for a page reload if they have forgotten to enter something.
  • Incerased interactivity – we can create interfaces that react when the user hovers. overthem with a mouse or actives them via the keyboard.
  • Richer interfaces – we can use javascript to include such item, drag and drop compoment & slide to give rich interface to our site visitor.

Condional Statements

Condional statements are used to make dicision out of a condition. The major condional statement are:-

1.if statement

2.if else statement

3.switch statement

1.if statement

Syntax:-

if(condition)
			{
			code to be execuated.if the condition is true
			}

eg:

if(a>b)
			{
			document.write("a is greater");
			}

2.if else statement

Syntax:-

if(condition)
			{
			code to be execuated if the condition is true
			}
			else
			{
			code to be executed "if the condition is false"
			}

eg:

if(a>b)
			{
			document.write("a is greater");
			}
			else
			{
			document.write("b is greater");
			}

Python

Python is a simple, general purpose, high level, and object-oriented programming language. Python supports multiple programming pattern, including object-oriented, imperative, and functional or procedural programming styles. Multipurpose programming language We don't need to use data types to declare variable because it is dynamically typed so we can write a=10 to assign an integer value in an integer variable. Python makes the development and debugging fast because there is no compilation.

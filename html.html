HTML

Hypertext markup language

It is the hypertext markup language. HTML is used for the creation of web pages html was created by Tim Berners-Lee in 1991. The first markup language used was GML (Genarlized markup language) and was invented by IBM later SGML(Standard GML) was invented by ISO.

later SGML was extended to form HTML and XML (extensible markup language). This was done by W3C.The latest version of HTML is HTML5.

Basic Syntax

The fundametal syntaxtic unit of HTML are called tag. In general tags are used to specify catagerics of condent. The syntax of a tag is the tags name surrounded by angle brackets () tag names be written in all lowercases. most tag appear in pairs. An opening tag & a closing tag. The name of a closing tag is the name of its corresponding opening tag with a slash attached to the begining.

eg:<p>&</p>

what ever appers between a tag and its corresponding of the closing tag is the condent of tax The opening tag & its closing tag together specify a condiner of the condent. they enclose the condiner and its condent together are called element.

eg:<p>hai</p>

This repeasent a paragraph element attributes are used to specify alternative meanings of a tag appers between an opening tag’s name and its right angle bracket. They are specified in keyword form which means attributes name its followed by an equal sign and an attribute value delimited by “”.

Table
The html tables allows us to arrange datas like text,images,links, other tables etc into rows and colums. The html tables are created using the table tag in which “<tr>”is used to create table rows and “<td>” is used to create data cells. The elements under “<td>” are regular and left aligned by default

eg:

<html>
<head>
</head>
<body>
<table border = "1">
<tr>
<td>row1 colum1</td>
<td>row1 colum2</td>
</tr>
<tr>
<td>row2 colum1</td>
<td>row2 colum2</td>
</tr>
</table>

Output

row1 colum1 row1 colum2
row2 colum1 row2 colum2

Frames in html

html frames are used to divided our browser window into multiple section. where each section can load a seprated html document. a collection of frames in the broswer is know as frameset. the window is divided into frames in a simaliar the tables are organised into rows & columns.

Disadvantages of frames

There are few drawbacks using frames. they are :-

1.some smaller devices can’t cope with frames often because there screen size is not big enough to be dividedup

2. some times our page will be defined differently on different computer due to different screen resoluation

3.broswer’s back button minue not work has user hope

4.there are still few broswer that do not support frame technology

Creating frames

To use frames on a page we use “<frameset>” instread of “<body>” frameset define how to divide window into frames.

eg:

<html>
<head>
<title>frames</title>
</head>
<frameset rows = "10%,80%,10%">
<frame name = "top" src ="/html/topframe.html"/>
<frame name = "bottom" src ="/html/bottomframe.html/">
<no frame>
<body>your browser do not support frames</body>
</no frame>
</frameset>
</html>

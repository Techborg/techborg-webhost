Advantages of CSS

2.Pages load faster:- for CSS there is no need to write HTML tag attribute everytime just write one CSS rule of a tag and apply it to all the occurcies of that tag so less code mean faster download time

3.Easy Maintances:-To make a globale simply change style,All element in page will be automatically updated

4.Surprior style to HTML:-CSS has much wider array of attributes than html so we can give a far better look to our html page

5.Multiple device combaitily:-style sheet allows content to be optizesemed for more than one type of device by using same html document, different versions of web site can be presented hand held devices such as mobile phones

6.Golable web standand:-now html attribute depreced and its is recommed to use CSS

CSS Syntax

A CSS Comprosses og style rules that are inteppurted by broswer and then applied to corrponding element in our html

1.selector- A selector is an html tag at which style is applied

eg: h1,table,p etc

2.properly- A properly is a type of attribute of HTML tag or CSS attribute. all the html attribute are automallically converted to CSS attribute

eg: color,border… etc

3.value- value are asigned to properties

eg: color:red

Different ways of define selectors

1.Type selector method

table{
			border:1px;
			}

2.Universal selector method

*{
			color:yellow;
			}

Rather than selecting element of a specific type universal selector is applicable every element inside the html document

3.Descendant selector method

ul p{
			color:red;
			}

suppose we want applies a style rule a particular element only when it comes under a particular tag. we use descendant selector method

4.Class selector method

we can define style rules based on class attribute of the element. all the elements having that class will be formated recording to define rule

black{
			color:red
			}
			<p class="black">
			<h1 class="no color">
			<p class="black">
			<h3 class="black">

to be more specific we can write

p.black{
			color:red
			}

Inclusion of CSS in HTML Document

There are four ways to include CSS into HTML document.There are:-

1.Embedded CSS

We can put our CSS rule into an HTML document using style tag <style> this tag is placed inside a head tag of the HTML document

<html>
			<head>
			<style type="text/css">
			h1{
			color:red;
			}
			</style>
			</head>
			<body>
			<h1>Hello<h1>
			</body>
			</html>

2.Inline CSS

We can use style attribute of any html element to define style rule. There rules will be appliable to that element only

<html>
			<head>
			</head>
			<body>
			<h1 style="{color:red;}">Hello</h1>
			</body>
			</html>

3.Exterenal CSS

<link> can be used to include an external CSS file to our HTML Document. an external CSS file is a seprated text file with CSS file, we can define all the style rules with in the text file and then we can included this file,in any HTML document using <link>

eg:consinder an external css file called mystyle.css

h1{
			color:red;
			}

now we can include mystyle.css file as shown below

<html>
			<head>
			<link type="text/css" href="mystyle.css"/>
			</head>
			<body>
			<h1>Hello</h1>
			</body>
			</html>

4.Imported CSS

@ import is used to import an external style sheet in a manner similar to <link> element

<html>
			<head>
			@import"mystyle.css"
			</head>

Multiple Style Rules

We may need to define multiple style rule for single element

h1{
			color:red
			font-size:20pt;
			}

Grouping selector

We can applys a style to many selectors

p,h1,h2,h3{
			color:red;
			font-size:20pt;
			}

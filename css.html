CSS (Cascading Style Sheat)


CSS is used to control the style of a web document in a simple and easy way. It is a simple design language indened to simply the process of making web page presentable CSS handles look and feel part of a web page. using css we can control colour of test the style of front, spacing between paragraphs background images layout design and varity of other effect our current version of css in CSS3


Advanages of CSS

1. CSS saves time :- we can write css onces and then reuse same style in multiple HTML pages.


2. Pages load faster :- for css there is no need to write HTML tag attribute every time just write one css rule of a tag and apply it to all the occurcies of that tag. so less code mean faster download time.


3. Easy Maintances :- To make a globale simply change style, All element in page will be automatically updated.


4. Surprior style to HTML :- CSS has much wider array of attributes than html so we can give a far better look to our html page.


5. Multiple device compatibility style sheet allows content to be optizesemed for more than one type of device by using same html document, differenent, differenent versions of website can be presented hand held devices such as mobile phones.


6. Globale web standard :- now html attribute depreced and its is recommed to use CSS. CSS is created and maiantance by W3C (World Wide Consortium)


CSS Syntax


A CSS comprosses of style rules that are inteppurted by broswer and then applied to corresponding element in our html.


A style rule is make up of three parts :-


1. Selector - A selector is an html tag at which style is applied.


eg: h1, table, p, etc


2. Properly - A properly is a type of attribute of html tag or CSS attribute. all the html attribute are automatically converted to CSS attribute


eg: color, border...etc


3. Value - A value are asigned to properties.


eg: color: red
